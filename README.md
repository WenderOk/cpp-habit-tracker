# Трекер привычек
Простой TUI трекер привыяек написанный на C++.
## Бэклог
Высший приоритет:
- Удаление/добавление привычек
  Создание объекта привычки с названием, частотой повторения, числовым или булевым флагом 
  (привычки задающие вопрос да/нет или сколько?).
- Хранение данных о привычках в файлах для сбора статистики. Сохранение и чтение данных из файлов.
- Расписание привычек на неделю
  отображение текущей недели и привычек на каждый день недели
Средний приоритет  
- Интерфейс в терминале (TUI)
  Главное меню со списком текущих привычек и кнопками для создания/удаления привычек
  и перехода на вкладку со статистикой привычки
- Система уведомлений
  отображение напоминаний о привычках через средства операционной системы
Низкий приоритет
- Система статистики	
  Кнопка для перехода на вкладку со статитстикой привычки, отображающую максимальное количество
  серий без пропусков и прочие метрики
